Création de Playlist

La création de playlist permet de renseigner plusieurs paramètres contenant des informations sur la personnalisation de sa veille informationnelle.

Méthode: POST

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/create

Paramètres :

  • id_playlist
    • Type String
    • Obligatoire
  • id_user
    • Type String
    • Obligatoire
  • nom
    • Type String
    • Obligatoire
  • categoriesContent
    • Type String
    • Obligatoire
  • celebritesContent
    • Type String
  • motsCleContent
    • Type String

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/create?
motsCleContent=football,cdm,bresil,psg
&id_playlist=22
&id_user=Neymar
&celebritesContent=Messi,Nasser,Suarez
&nom=NeymarSportPlaylist
&categoriesContent=Sports

Informations reçues en retour de l’appel :

  • Succès : "Resultat": "New Playlist : nomPlaylist is created"
  • Erreur (si id_playlist existe déjà) : “Error” : This id playlist is already exist

Modification de Playlist

La modification de playlist permet de changer les informations liées à une playlist à l’aide de l’id_playlist dans la base de données.

Méthode: PUT

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/update

Paramètres :

  • id_playlist
    • Type INT
    • Obligatoire
  • nom
    • Type String
  • categoriesContent
    • Type String
    • Par défaut toutes les catégories sont ajoutées
  • celebritesContent
    • Type String
  • motsCleContent
    • Type String

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/update?
motsCleContent=tesla,voiture,fusée&
id_playlist=22&
id_user=ElonMusk&
celebritesContent=BilleGate,MarkZuckerberg&
nom=ElonMuskPlaylist&
categoriesContent=Technology,business

Informations reçues en retour de l’appel :

  • Succès : Affichage de la playlist à jour sous forme de JSON
  • Erreur : Something wrong when updating data !

Suppression de Playlist

La suppression de playlist permet de supprimer une playlist de la base de donnée grâce à l’id_playlist.

Méthode: DELETE

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/delete

Paramètres :

  • id_playlist
    • Type INT
    • Obligatoire

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/delete?id_playlist=22

Informations reçues en retour de l’appel :

  • Succès : “Resultat : Playlist *id_playlist* deleted”
  • Erreur : “Error : Playlist *id_playlist* is not deleted”

Recherche de Playlist

La recherche d’une playlist est possible à l’aide de l’id_playlist ou du nom en parcourant la base de données. En cas de doublons de nom, toutes les playlists s’afficheront.

Méthode: GET

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/search

Paramètres :

  • id_playlist
    • Type INT
    • Obligatoire / Optionnel
  • nom
    • Type String
    • Obligatoire / Optionnel

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/search?id_playlist=22&nom=MyPlaylist

Informations reçues en retour de l’appel :

  • Succès : Affiche le(s) JSON correspondant “Playlist :{...}”
  • Erreur : "error": "No playlist found in database"

Inscription Utilisateur

L'inscription sert à créer la fiche de l'utilisateur, il y renseigne son nom, prénom, pseudo, mot de passe, adresse email, sa langue, ses différents abonnements à d'autres utilisateurs et aussi son fil d'actualité.

Méthode: POST

Route: https://api-playlist-veille-ecv.herokuapp.com/api/inscription

Paramètres :

  • nom
    • Type String
    • Obligatoire
  • prenom
    • Type String
    • Obligatoire
  • pseudo
    • Type String
    • Obligatoire
  • password
    • Type String
    • Obligatoire
  • email
    • Type String
  • abonnement
    • Type list d'utilisateur
    • (facultatif)
  • lang
    • Type String
    • Defaut: "fr"
    • (facultatif)
  • fil_actu
    • Type String

Exemple d'utilisation de la création :

https://api-playlist-veille-ecv.herokuapp.com/api/inscription?nom=votrenom&
prenom=votreprenom&
pseudo=votrepseudo&
email=votreadresseemail&
password=votrepassword&
abonnement=vosabonnement&
lang=fr&
fil_actu=voscategoriesdufildactualite

Informations reçues en retour de l’appel :

  • Succès : "Resultat": "New user create"
  • Erreur : Si le pseudo et/ou l'adresse email est déjà dans notre base de données
    error "This user pseudo already existe"
    error "This user email already existe"

Connexion Utilisateur

La connexion permet aux utilisateurs d'avoir accès aux informations déployé par l'API. Une fois connecté à celle-ci, l'utilisateur retrouve son fil d'actualité qu'il a paramétré lors de son inscription, il peut également modifier ou supprimer son compte.

Méthode: GET

Route: https://api-playlist-veille-ecv.herokuapp.com/api/connexion

Paramètres :

  • email
    • Type String
    • Obligatoire
  • password
    • Type String
    • Obligatoire

Exemple d'utilisation de la connexion' :

https://api-playlist-veille-ecv.herokuapp.com/api/connexion?email=adresseemail&password=votremotdepasse

Informations reçues en retour de l’appel :

  • Succès : "Resultat" :
    • Information de l'utilisateur (au format json)
    • Liste des news de son fil d'actualité (au format json)
  • Erreur : "Incorrect pseudo or password"

Modification Utilisateur

La modification sert aux utilisateurs à remplacer leurs informations, le nom, le prénom, le pseudo,le mot de passe,l’adresse email, l’abonnement, la langue et également le fil d’actualité.

Méthode: PUT

Route: https://api-playlist-veille-ecv.herokuapp.com/api/update

Paramètres :

  • nom
    • Type String
    • Obligatoire
  • prenom
    • Type String
    • Obligatoire
  • pseudo
    • Type String
    • Obligatoire
  • password
    • Type String
    • Obligatoire
  • email
    • Type String
  • abonnement
    • Type list d'utilisateur
    • (facultatif)
  • lang
    • Type String
    • Defaut: "fr"
    • (facultatif)
  • fil_actu
    • Type Array
    • les valeurs doivent être séparé par une virgule “,”
    • valeurs : business entertainment general health science sports technology

Exemple d'utilisation de la modification' :

https://api-playlist-veille-ecv.herokuapp.com/api/update?nom=votrenom&
prenom=votreprenom&
pseudo=votrepseudo&
email=votreadresseemail&
password=votrepassword&
abonnement=vosabonnement&
lang=fr&
fil_actu=voscategoriesdufildactualite

Informations reçues en retour de l’appel :

  • Succès : "Resultat": JSON du user
  • Erreur : "Something wrong when updating data!"

Suppression Utilisateur

La suppression sert à effacer toutes les données de la base de données d'un utilisateur. Une fois faite avec succès, il n'y a aucun moyen de récupérer les données supprimées.

Méthode: DELETE

Route: https://api-playlist-veille-ecv.herokuapp.com/api/delete

Paramètres :

  • email
    • Type String
    • Obligatoire

Exemple d'utilisation de la suppression :

https://api-playlist-veille-ecv.herokuapp.com/api/update?email=votreadresseemail

Informations reçues en retour de l’appel :

  • Succès : "User delete"
  • Erreur : "error: err"

Recherche Utilisateur

La recherche d'un utilisateur sert à afficher les données d'un utilisateur. La recherche s'effectue à l'aide du nom, prénom, pseudo ou email. S'il y a plusieurs utilisateurs avec le même nom ou prénom, tous ces utilisateurs seront affichés. Un seul paramètre est accepté.

Méthode: GET

Route: https://api-playlist-veille-ecv.herokuapp.com/api/user

Paramètres :

  • nom
    • Type String
    • Obligatoire
  • prenom
    • Type String
    • Obligatoire
  • pseudo
    • Type String
    • Obligatoire
  • email
    • Type String
    • Obligatoire

Exemple d'utilisation de la recherche :

https://api-playlist-veille-ecv.herokuapp.com/api/user?pseudo=unpseudo

Informations reçues en retour de l’appel :

  • Succès : Affichage du JSON contenant les informations de l'utilisateur.
  • Erreur : "No user found in database"

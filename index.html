Création de Playlist

La création de playlist permet de renseigner plusieurs paramètres contenant des informations sur la personnalisation de sa veille informationnelle.

Méthode: POST

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/create

Paramètres :

  • id_playlist
    • Type String
    • Obligatoire
  • id_user
    • Type String
    • Obligatoire
  • nom
    • Type String
    • Obligatoire
  • categoriesContent
    • Type String
    • Obligatoire
  • celebritesContent
    • Type String
  • motsCleContent
    • Type String

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/create?
motsCleContent=football,cdm,bresil,psg
&id_playlist=22
&id_user=Neymar
&celebritesContent=Messi,Nasser,Suarez
&nom=NeymarSportPlaylist
&categoriesContent=Sports

Informations reçues en retour de l’appel :

  • Succès : "Resultat": "New Playlist : nomPlaylist is created"
  • Erreur (si id_playlist existe déjà) : “Error” : This id playlist is already exist

Modification de Playlist

La modification de playlist permet de changer les informations liées à une playlist à l’aide de l’id_playlist dans la base de données.

Méthode: PUT

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/update

Paramètres :

  • id_playlist
    • Type INT
    • Obligatoire
  • nom
    • Type String
  • categoriesContent
    • Type String
    • Par défaut toutes les catégories sont ajoutées
  • celebritesContent
    • Type String
  • motsCleContent
    • Type String

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/update?
motsCleContent=tesla,voiture,fusée&
id_playlist=22&
id_user=ElonMusk&
celebritesContent=BilleGate,MarkZuckerberg&
nom=ElonMuskPlaylist&
categoriesContent=Technology,business

Informations reçues en retour de l’appel :

  • Succès : Affichage de la playlist à jour sous forme de JSON
  • Erreur : Something wrong when updating data !

Suppression de Playlist

La suppression de playlist permet de supprimer une playlist de la base de donnée grâce à l’id_playlist.

Méthode: DELETE

Route: https://api-playlist-veille-ecv.herokuapp.com/api/playlist/delete

Paramètres :

  • id_playlist
    • Type INT
    • Obligatoire

Exemple de création de playlist :

https://api-playlist-veille-ecv.herokuapp.com/api/playlist/delete?id_playlist=22

Informations reçues en retour de l’appel :

  • Succès : “Resultat : Playlist *id_playlist* deleted”
  • Erreur : “Error : Playlist *id_playlist* is not deleted”
